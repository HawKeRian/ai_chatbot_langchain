# AI Chatbot Template

### Description:
The mock-up QC Assistant, an expert Quality Control Analyst specializing in root cause analysis and production auditing.
Connect with database and external sources by using custom tools.

### Tools & Libraries
- Python 3.13
- Oracle Database Connection
- Langchain
- Langgraph
- Chainlit
- OpenAI Model: gpt-5.2 (with Azure)
